# 상품 목록 페이지 리뷰

## 고객사 요구사항

1. 젊은 세대가 주로 모바일을 사용하기 때문에 모바일 환경에서의 사용성을 중요하게 생각합니다.
2. 신발의 사진이 디자인 시안처럼 잘 보이게 작업해주세요.
3. 장바구니에 상품을 추가할 때, 헤더에서 실시간으로 상품 개수가 표시되도록 해주세요.
4. 작업 결과물을 확인할 수 있는 테스트 URL을 제공해주세요.
5. 상품 카드는 2열 구조로 구성
6. 상품 설명은 최대 2줄까지 표시되며, 초과 시 말줄임 처리
7. “담기” 버튼 클릭 시 상태 변경(담기 ↔ 해제)
8. 동일 상품 중복 추가 방지
9. 직관적인 버튼 디자인 적용

## 요구사항에 따른 기능 동작 확인

### 1. 모바일 환경 구성

- 모바일 화면 기준으로 2열 고정 레이아웃 적용
- 한 손 조작이 편리하도록 카드 및 버튼 크기 최적화
- iOS Safari, 삼성 브라우저 등 주요 모바일 브라우저에서 정상 동작 확인
- 세로 스크롤 중심 구조로 구성하여 탐색 편의성 강화

적용 결과
- 모바일에서 가독성과 터치 편의성이 확보됨
- 상품 탐색 흐름이 자연스럽고 끊김 없이 작동함

### 2. 신발 상품 사진

- 상품 이미지가 카드 상단에 크게 배치되도록 구성
- 이미지 비율이 깨지지 않도록 고정 비율 유지
- 시안과 동일한 여백 및 카드 디자인 적용
- 다양한 해상도 환경에서도 선명하게 표시

적용 결과
- 신발 디자인이 한눈에 들어오는 구조
- 이미지 왜곡 없이 안정적으로 노출됨

### 3. 장바구니 담기 기능

각 상품 카드에 “담기” 버튼 배치
- 버튼 클릭 시 즉시 “해제”로 상태 변경
- 다시 클릭 시 장바구니에서 제거
- 동일 상품은 중복 추가되지 않음
- 버튼 상태 변화가 시각적으로 명확하게 표시됨

적용 결과
- 사용자 행동에 즉각적인 피드백 제공
- 실수로 여러 번 담는 상황 방지

### 4. 헤더 실시간 상품 개수 표시

- 장바구니에 상품을 추가하면 상단 헤더의 아이콘 옆 숫자가 즉시 증가
- 상품을 해제하면 숫자도 즉시 감소
- 페이지 새로고침 없이 실시간 반영

적용 결과
- 사용자가 현재 장바구니 상태를 즉시 확인 가능
- 쇼핑 흐름이 끊기지 않음

### 5. 상품 카드 구성

- 2열 고정 레이아웃
- 카드 내부 구성:
    - 상품 이미지
    - 브랜드명
    - 상품 설명 (2줄 제한 + 말줄임 처리)
    - 가격
    - 담기 버튼
- 카드 간 일정한 간격 유지
- 디자인 시안과 동일한 구조 반영

### 6. 사용자 경험(UX) 고려 사항

- 버튼 크기 및 간격을 모바일 터치 기준에 맞춰 설계
- 시각적 구분이 명확한 색상 사용
- 과도한 정보 노출을 피하고 핵심 정보 중심 구성
- 빠른 로딩과 부드러운 스크롤 환경 유지
